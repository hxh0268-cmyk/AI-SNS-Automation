# Image Generation Prompt

## Specifications

- Platform: Instagram
- Aspect ratio: 1:1 (square)
- Target audience: Restaurant managers and owners
- Style: Simple, clean, minimal text, AI-driven feel
- Design: Japanese-friendly aesthetic
- Slide type: CTA

## Prompt

1:1 square image. A clean, modern, and action-oriented design for an Instagram carousel CTA (Call To Action) slide targeting restaurant owners. The main Japanese text 'この投稿を保存して明日から投稿時短！' (Save this post and shorten your posting time from tomorrow!) is prominently displayed in a large, clear, and inviting Japanese font, centrally placed. The background features a prominent, stylized 'save' icon (e.g., a modern bookmark or a digital folder icon) subtly integrated with dynamic, glowing arrows pointing towards the save action. The design is simple, direct, and highly encouraging, prompting immediate action. The color palette predominantly features cool blues and greens, complemented by clean whites and subtle, modern digital textures. This color scheme and texture ensure visual consistency with the other slides in the carousel. Icon and text colors are harmonized with this cool-toned palette. High-tech, AI-generated aesthetic. Japanese design aesthetic.
